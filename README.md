# Resolve+

Sistema de gerenciamento de reclamações desenvolvido para a atividade A3 da disciplina de Interação Humano-Computador (IHC) e User Experience (UX).

## Objetivo

O Resolve+ foi criado para centralizar e organizar reclamações de clientes, permitindo o registro, acompanhamento e gerenciamento de chamados de forma simples e intuitiva.

## Funcionalidades

- Cadastro de reclamações
- Consulta automática de endereço via API ViaCEP
- Registro de loja ou estabelecimento relacionado à reclamação
- Listagem dinâmica de reclamações
- Visualização detalhada dos chamados
- Alteração de status (Aberta, Em análise e Resolvida)
- Exclusão de reclamações
- Dashboard com indicadores em tempo real
- Estatísticas por categoria e status
- Interface responsiva para desktop, tablet e mobile

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- LocalStorage
- API ViaCEP
- GitHub Pages
