Sobre o Resolve+

Sistema web interativo para gestão inteligente de reclamações.

Objetivo

Organizar reclamações de clientes, permitindo visualizar status, categorias, prioridades e indicadores de atendimento.

Tecnologias

HTML para estrutura, CSS para interface responsiva e JavaScript para consumo da API ViaCEP.

API Utilizada

O sistema usa a API ViaCEP para buscar automaticamente endereço, bairro, cidade e estado a partir do CEP informado.

UX e IHC

Interface com contraste adequado, navegação clara, feedback visual, cards organizados e layout adaptável para diferentes telas.
